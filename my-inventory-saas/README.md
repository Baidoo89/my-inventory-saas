## ?? Project Overview
StockFlow is a cloud-based Inventory Management and Point of Sale (POS) system designed for Small and Medium Enterprises (SMEs). It solves the problem of manual stock keeping by providing a digital, real-time solution for tracking products, recording sales, and managing business performance.

## ? Key Features
*   **Dashboard Analytics**: Real-time overview of sales, revenue, and low stock alerts.
*   **Inventory Management**: Add, edit, and delete products with automatic stock tracking.
*   **Point of Sale (POS)**: Streamlined checkout interface for processing sales quickly.
*   **Subscription System**: SaaS model with 'Starter' and 'Pro' plans (includes expiration enforcement).
*   **Offline Capability**: Continue selling even when the internet connection is lost (syncs when back online).
*   **Responsive Design**: Fully functional on Desktops, Tablets, and Mobile phones.

## ??? Technology Stack
*   **Frontend**: Next.js 16 (React Framework)
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS
*   **Backend/Database**: Supabase (PostgreSQL + Auth)
*   **State Management**: React Hooks
